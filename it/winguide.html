Guida dell'Apocalitticraft Installer

Versione 1.2
Home

Installazione

  • Scarica il file
  • Apri il setup
  • Clicca "Install Apocalitticraft"

Aggiornamento

  • Apri il setup
  • Clicca "Update Apocalitticraft"

Estrai un launcher

  • Apri il setup
  • Scegli il launcher da estrarre
  • Spunta la casella se vuoi estrarlo nella cartella del setup, altrimenti verrà estratto in .minecraft
  • Clicca "Extract launcher"

Disinstallazione

  • Apri il setup
  • Clicca "Uninstall Apocalitticraft"

Errori

  • 01 - Install Minecraft 1.8.8 First (No MC 1.8 assets found)

    • Se vedi questo errore, installa Minecraft 1.8.8. Se non hai un launcher, puoi estrarlo con il setup. Una volta installato, clicca su "refresh installer"

  • 02 - Error during uninstall. Wait a minute and try again

    • Se vedi questo errore, chiudi APC e aspetta. Se non risolvi, vai "%appdata%\.minecraft\versions\apc" e cancella la cartella natives.

  • 03 - Unable to install

    • Questo errore non dovrebbe più apparire
