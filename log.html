1st Januray 2018, v0.2

Hi all! We finally decided the name and the theme of our mod! It's official name is Apocalitticraft and it will be a post-apocalittic mod and the target will be survive in this world destroyed by wars, bombs and radiations. We planned a lot of changes like new items, new mobs and also a red sky! You'll see this changes in the next weeks or maybe next months.

With this Alpha version we also provide an all new installer built in Visual Basic with new options, graphics and more.
