Home

Install

  • Download the file
  • Open setup
  • Select "Install"

Update

  • Open setup
  • Choose "Update"

Extraxt launcher only

  • Open setup
  • Select what launcher you want to install
  • Check the box if you want to extract launcher in the Setup dir (Leave it empty if you want the launcher in .minecraft dir)
  • Select "Extract launcher"

Uninstall

  • Open setup
  • Select "Uninstall"

Errors

  • 01 - Install Minecraft 1.8.8 First (No MC 1.8 assets found)

    • If you see this message, open a launcher (if you don't have it, extract from the setup) and download a Minecraft 1.8 version. Then refresh the setup.

  • 02 - Error during uninstall. Wait a minute and try again

    • If you see this error, run Minecraft APC, close it and wait a minute. Then try again. If the error persists, go to "%appdata%\.minecraft\versions\apc" and delete the natives folder.

  • 03 - Unable to install

    • This error is teorically fixed.
